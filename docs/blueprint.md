# **App Name**: Ingredient Insights AI

## Core Features:

- Ingredient Text Input: Accept ingredient lists via scan or paste.
- Ingredient Analysis: Send ingredient list to the backend for analysis and insights.
- AI-Powered Summary Generation: Generate a concise summary of the ingredient list analysis.
- Ingredient Highlight and Explanation: Identify and highlight potentially concerning ingredients with AI-powered explanations.
- Confidence Scoring: Add a confidence level to each potentially concerning ingredient interpreted by the AI tool.
- Uncertainty Display: Provide clear indicators of uncertainty in the AI analysis.
- Actionable Suggestions: Suggest relevant next steps based on the ingredient analysis.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey intelligence and trustworthiness.
- Background color: Dark navy (#1A237E) to provide a sleek, modern feel and contrast with UI elements.
- Accent color: Electric purple (#BE95FF) to highlight key information and action items, analogous to the indigo primary.
- Body and headline font: 'Inter' for a modern, machined look.
- Minimalist layout with rounded cards and glassmorphism-style panels to create a clean, futuristic interface.
- Simple, minimalist icons to represent different ingredient categories and concerns.
- Calm AI-style loading animation (no spinners).
